# 五子棋游戏 🎯

一个完全独立运行的五子棋游戏，可直接在浏览器中玩耍！

## 功能特性

✨ **完整的游戏功能**
- 15×15 棋盘
- 人机对战（AI vs 玩家）
- 实时游戏状态显示
- 步数计数

🤖 **智能 AI 对手**
- 自动防守（阻止玩家获胜）
- 主动进攻（寻求获胜机会）
- 策略性落子（优先中心区域）
- 位置评估算法

🎮 **游戏控制**
- **落子**：直接点击棋盘
- **悔棋**：撤销最后两步
- **重新开始**：清空棋盘，重新游戏
- **认输**：主动放弃比赛

📱 **响应式设计**
- 支持桌面和移动设备
- 自适应屏幕大小
- 优美的视觉设计

## 使用方法

### 方式一：直接打开 HTML 文件

1. 下载 `gobang.html` 文件
2. 用浏览器打开（Chrome、Firefox、Safari、Edge 都支持）
3. 开始游戏！

### 方式二：在线体验

将 `gobang.html` 上传到 GitHub Pages，即可在线玩耍。

## 游戏规则

- 黑棋（玩家）先手
- 白棋（AI）后手
- 首先连成 5 个或以上的棋子（横、竖、斜）的一方获胜
- 支持悔棋、认输等操作

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计与响应式布局
- **JavaScript (ES6+)** - 游戏逻辑
- **Canvas API** - 棋盘绘制

## 代码结构

```
GobangGame 类
├── init()           // 初始化游戏
├── handleClick()    // 处理玩家点击
├── aiMove()         // AI 走棋
├── getBestMove()    // 计算最优步数
├── checkWin()       // 检查胜负
├── draw()           // 绘制棋盘
└── 其他辅助方法
```
